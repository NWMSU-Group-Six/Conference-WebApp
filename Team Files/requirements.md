## Requirements

**Fr-1:** As an admin, I want to create and manage basic conference information including dates, venue, theme, and description, so that participants can learn about the conference.
Priority: 1

**Fr-2:** As a user, I want to view conference information including calls for papers, submission deadlines, registration details, and program schedules on the homepage, so that I can quickly access critical information. 
Priority: 1 

**Fr-3:** As an admin, I want to create and publish announcements and news updates, so that I can keep participants informed of important changes or reminders.  

**Fr-4:** As an admin, I want to be able to send announcements to participants/sponsors.  

**Fr-5:** As an admin, I want to restrict administrative features to authorized users so that system integrity is maintained.
Priority: 1

**Fr-6:** As a user, I want to create an account with email verification and login securely, so that I can register, submit papers, and manage my conference information.  
Priority: 1

**Fr-7:** As a user, I want to reset my forgotten password through an automated email process, so that I can regain access to my account.  
Priority: 1

**Fr-8:** As a sponsor, I want to be able to contact conference organizers to become a sponsor for the conference.  
Priority: 2 

**Fr-9:** As an admin, I want to assign reviewers to paper submissions, so that all papers receive appropriate evaluation.  
Priority: 1

**FR-10:** As a reviewer, I want to access assigned paper submissions to evaluate them, so that I can fulfill my review responsibilities.  
Priority: 1

**FR-11:** As a reviewer, I want to submit scores and comments for paper submissions, so that authors and organizers receive feedback.  
Priority: 1

**Fr-12:** As an author, I want to be able to withdraw or edit my submission
Priority: 2

**Fr-13:** As an author, I want confirmation of my submission so that I know it was received successfully.
Priority: 2

**FR-14:** As a user, I want to register for the conference by providing my information including name, affiliation, email, and dietary restrictions, so that I can secure my attendance.  
Priority: 1

**FR-15:** As a user, I want to receive registration confirmation emails with receipt and conference details, so that I have documentation of my registration.  
Priority: 2
